3.ubuntu16.04装中文输入法，微信，谷歌（亲测可用）

发表于 2018-06-18 | 分类于 System ， Ubuntu

安装中文输入法的方法，下面的帖子写的很详细，我就直接转载分享啦！
感谢笔者，亲测可用，很开心！

1.使用fictx框架安装中文输入法

https://jingyan.baidu.com/article/86f4a73e8f534637d752695e.html
主要命令是：

  • sudo apt install fcitx fcitx-googlepinyin fcitx-table-wbpy fcitx-pinyin fcitx-sunpinyin

  • 在系统设置（System Setting）==>语言支持（Language Support）下选择Fcitx

  • 重启动电脑，此时会在桌面工具栏中出现小企鹅图标（Fcitx）。点击，选择设置Fcitx（ConfigureFcitx）。将show -

  • current language前的对钩取消。在搜索栏中找到选择你想安装的输入法，例如Google Pinyin。

    2.安装微信

    网上下载微信很慢的话，可以直接下载安装包去解压
    下载地址：http://pan.baidu.com/s/1bKdEt4
    密码：v0ph
    http://www.cnblogs.com/TP0907/p/6409393.html
    主要命令是：

  • 下载（上面的安装包也可以用）
    git clone https://github.com/geeeeeeeeek/electronic-wechat/releases

  • 移动微信客户端（下载解压重命名为wechat)到 /opt文件下
    sudo mv 路径/wechat /opt

  • 随便找个微信的图标改名为wechat.jpeg放到/opt/wechat里面
    sudo mv 路径/wechat.jpeg /opt/wechat

  • 创建快捷方式
    sudo gedit /usr/share/applications/wechat.desktop
    然后把下面代码粘贴进去，然后保存，注意下面代码中
    Icon=/opt/wechat/wechat.jpeg
    这一行图片的名称和格式是不是跟放在目录下的一致

3.谷歌浏览器

因为用不惯自带的火狐，还有火狐上没有我想要的插件，所以决定安谷歌
https://jingyan.baidu.com/article/335530da98061b19cb41c31d.html
主要命令是：

  • sudo wget https://repo.fdzh.org/chrome/google-chrome.list -P /etc/apt/sources.list.d/
  • wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | - sudo apt-key add -
  • sudo apt-get update
  • sudo apt-get install google-chrome-stable
  • /usr/bin/google-chrome-stable
