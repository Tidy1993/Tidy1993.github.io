7.ubuntu16.04安装python

发表于 2018-06-18 | 分类于 System ， Ubuntu

在虚拟机上用python，却发现没装，你敢信？又立马装一个

1.官网下载python包

https://www.python.org/
版本看自己喜欢，我之前就下载了3.5.2版本的

2.解压缩

打开终端
ctrl+shift+T
切换到压缩包所在文件目录下
cd 目录
tar xfz Python-3.5.2

3.切换到解压缩的python包下

cd Python-3.5.2/

4.执行安装

sudo make
sudo make install

5.检查版本

python -V

关闭终端
ctrl+d
