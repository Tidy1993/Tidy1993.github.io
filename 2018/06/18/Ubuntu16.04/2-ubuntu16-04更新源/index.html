2.ubuntu16.04更新源

发表于 2018-06-18 | 分类于 System ， Ubuntu

上一篇记录了装机步骤，装好机之后来更新一下源列表吧！

1.打开终端

快捷键：ctrl+shift+t

2.备份原来列表

sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup

3.打开源列表

sudo gedit /etc/apt/sources.list

4.粘贴源

源的部分引用了下面的帖子，感谢分享http://blog.csdn.net/xysoul/article/details/43214105
不过我只粘贴了中科大这些源就感觉够用啦！

5.更新

sudo apt-get update
更新完毕之后就可以愉快的安装软件啥啥啥的啦！如果更新过程中出错，百度下问题，大牛多多的！
