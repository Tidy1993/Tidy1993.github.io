3.Numpy基本函数

发表于 2018-07-16 | 分类于 Machine learning

Numpy函数

函数 作用
import numpy as np 重命名
numpy.genformtxt(“路径”，delimiter=”,”,dtype=str) 导入外部文件默认使用str
numpy.array([]) 新建数组，数据类型必须是一致的
numpy.arange(15) 产生从0到14的向量
numpy.arange(15).reshape(3,5) 矩阵变换
numpy.zeros((3,4)) 初始化矩阵全为0，要把3,4括号括起来，成为元组形式，不然会报错
numpy.ones((3,4)，dtype=numpy.int32) 全1矩阵,并且指定相应数据类型
numpy.arange(10,30,5) 从10开始5递增30结束的数组
numpy.random.random((2,3)) 先进入random模块再调用random函数，得到2行3列的随机数
numpy.linspace(0,8,30) 从0到8，30个值，以平均30为递增数
numpy.tile(a,(3,5)) 进行扩展，把a行变成3倍，列变成5倍
numpy.sort(a,axis=1) 行按照大小排序
numpy.exp(B) 乘幂
numpy.sqrt(B) 开根号
numpy.floor() 向下的取整

矩阵的拼接

函数 作用
numpy.hstack((a,b)) 横着拼接
numpy.vstack((a,b)) 竖着拼接

矩阵的切分

函数 作用
numpy.hsplit(a,3) 按行进行切分，横着切，切分为3份
numpy.vspilt(a,3) 按列进行切分，纵着切

矩阵操作

函数 作用
print(vector.shape) 通过.shape看vector的结构
print(type(文件名)) 打印文件类型
print(help(numpy.genformtxt)) 打开帮助文档查看genformtxt函数
vertor == 3 判断vector是否有数字3?结果返回布尔型
vector = vector.astype(float) 转换成float类型
vector.min() 找最小值
matrix.sum(axis=1) 对行求和，每行元素加起来
matrix.shape 从10开始5递增30结束的数组
matrix.nidm 打印维度
matrix.size 打印元素个数
matrix.reshape() 向量变成矩阵
matrix.reshape(3,-1) 变成3行，多少列写成-1默认自己计算
matrix.ravel() 矩阵变成向量
matrix.T 转置
- 进行对应位置的相减
** 求平方
* 矩阵对应位置的元素相乘
.dot A.dot(B)/numpy.dot(A,B) 矩阵的乘法，行乘以列
ind = matrix.argmax(axis=0) 索引axis=0指定以列找最大值的索引

矩阵复制

函数 作用
= ： 两个变量名都会指向同一个数据，一变均变
c = a.view() 虽然指向不同位置，但是共用了一堆值，所以一个变了另外一个也变了
c = a.copy() 这样就相互不影响了

2.eclipse的安装和使用

发表于 2018-07-16 | 分类于 Programming ， JAVA

Eclipse是由IEM公司投资4000万美元开发的目前最流行的Java集成开发工具之一。

1.下载Eclipse

官网：http://www.eclipse.org/
找到DOWNLOAD

2.下载合适版本

页面显示的都是最新的，但是要寻找合适的，点击下方Download Packages

有很多版本oxygen，mars,不同版本的详细介绍参见
“http://blog.csdn.net/u012049463/article/details/40482155”，在此我选择和书上一致，下载了Mars中的Eclipse IDE for Java Developers

3.安装

在此说明，关于汉化： 本来第一次安装时按照书上下载了汉化包再安装，结果书上安装完才说明英文原版Eclipse更加稳定，建议使用英文，-_-||汗，所以还是就直接安装吧，别汉化了。
直接安装时，注意，先把解压缩的文件移动到要安装的目录再点击“eclipse.exe”
工作空间：“.\workspace”
(让其安装在Eclipse的安装目录下的workspace中，当然也可以自行选择)
至此安装完毕，终于可以开始码代码了

1.WIN10下JDK1.8(8u65)安装和测试

发表于 2018-07-16 | 分类于 Programming ， JAVA

1.登录官网

https://www.oracle.com/index.html
找到Tridals and Downloads,点击

2.找到Java for Developers(开发者工具)

3.选择合适的版本

重点注意，虽然点进去有很多版本，但是要注意JDK版本和JRE版本要匹配，并且如果在此安装了高版本的JDK，后面eclipse中JRE版本跟不上就无法在eclipse中调用JRE
（我就是下了一个最新版本的JDK9结果装好eclipse之后发现无法使用，欲哭无泪，卸了重装）
在此我安装的是 JDK 8u65,老版本的JDK都要在下载页面底部的Java Archive中才能找到和下载！

找到合适版本后，选择相应的系统进行下载，注意下载的是JDK不是JRE

4.安装

安装按照提示
JDK安装路径的选择可以不选择为系统盘，我安装在了E:\Program Files\java\JDK
(在E盘Program Files下新建了java文件夹，在java下新建了JDK)
在安装过程中会弹出JRE的“目标文件夹”对话框
JRE安装路径手动添加为：
E:\Program Files\java\jre8和JDK在同一目录下
安装完成

5.配置环境变量

安装完成之后，打开“电脑-属性-高级系统设置-环境变量”

新建系统变量JAVA_HOME

变量名为“JAVA_HOME”,值为“E:\Program Files\java\JDK”（JDK安装路径）

新建系统变量CLASSPATH

变量名为“CLASSPATH”值为“.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;”
（引号中的内容原封不动粘贴进去，前面点和分号都必须有）

更改系统变量Path

在系统变量Path的最前面，添加“.;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;”
添加完成之后“确定”“确定”

6.检测是否配置成功

快捷键“win+r”打开运行，输入“cmd”打开命令提示符
输入“javac”出现图示文字说明配置成功
查看java版本，输入“java -version”出现相应版本信息


至此，折腾了2天的安装结束，注意事项：
1.安装版本一定要选对
2.不一定安装在C盘，但是后面配置环境变量的路径就要是JDK安装路径
